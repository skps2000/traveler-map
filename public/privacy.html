시행일자: 2026년 1월 9일

제1조 (개인정보의 처리 목적)

주식회사 엑스티아이(이하 "회사")는 다음의 목적을 위하여 개인정보를 처리합니다. 처리하고 있는 개인정보는 다음의 목적 이외의 용도로는 이용되지 않으며, 이용 목적이 변경되는 경우에는 개인정보 보호법 제18조에 따라 별도의 동의를 받는 등 필요한 조치를 이행할 예정입니다.

  1. 회원 가입 및 관리
    회원제 서비스 제공에 따른 본인 식별·인증
    회원자격 유지·관리
    서비스 부정이용 방지
    각종 고지·통지
    고충처리
  2. 서비스 제공
    맞춤형 영어 학습 콘텐츠 제공 (난이도, 카테고리별 표현 추천)
    학습 진행도 관리 (XP, 레벨, Streak, 배지, 통계)
    간격 반복 알고리즘 기반 복습 일정 생성
    학습 리마인더 푸시 알림 발송
    학습 데이터 분석 및 개인별 숙련도 측정
    햄스터 캐릭터 인터랙션 맞춤화
  3. 서비스 개선 및 신규 서비스 개발
    서비스 이용 통계 분석 (익명화된 데이터)
    사용자 행동 패턴 분석을 통한 UI/UX 개선
    신규 학습 콘텐츠 및 기능 개발
    서비스 품질 향상 및 오류 수정
  4. 마케팅 및 광고 활용
    이벤트 및 광고성 정보 제공 (사전 동의한 경우에 한함)
    서비스 이용 빈도 파악
    회원의 서비스 이용에 대한 통계

제2조 (처리하는 개인정보의 항목)

  1. 필수 수집 항목
    • 회원가입 (이메일): 이메일 주소, 비밀번호(암호화 저장) - Supabase Auth
    • 회원가입 (소셜 로그인): 소셜 계정 ID, 이메일 주소, 프로필 사진(선택) - Supabase OAuth
    • 서비스 이용: 학습 데이터(학습한 표현 ID, 정답/오답 기록, 학습 세션 기록), 게이미피케이션(XP, 레벨, Streak 일수, 배지 획득 내역), 진행도(표현별 숙련도, 정답률, 복습 일정), 접속 로그(IP, 일시, 기기 정보) - 앱 사용 중 자동 수집
  2. 선택 수집 항목
    • 사용자명(닉네임): 프로필 커스터마이징 목적
    • 프로필 사진: 사용자 선택 시 Supabase Storage에 저장
    • 푸시 알림 토큰: 학습 리마인더 발송을 위해 Expo Notifications 통해 수집
    • 알림 설정 시간: 사용자 맞춤 알림 발송 시간
  3. 자동 수집 항목
    • 기기 정보: 모바일 기기 모델명, OS 버전, 앱 버전
    • 네트워크 정보: IP 주소, 통신사 정보
    • 학습 행동 데이터: 학습 시간대, 학습 소요 시간, 카드 플립 횟수, 퀴즈 응답 시간
    • 앱 사용 로그: 화면 전환 기록, 오류 발생 로그(Crash Report)
    ※ 회사는 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 만 14세 미만 이용자는 법정대리인의 동의가 필요합니다.

제3조 (개인정보의 처리 및 보유 기간)

  1. 회원 정보
    보유 기간: 회원 탈퇴 시까지
    파기 시점: 회원 탈퇴 요청 후 지체 없이 파기
    예외: 관련 법령에 따라 보존이 필요한 경우 해당 기간 동안 별도 보관
    • 계약 또는 청약철회 등에 관한 기록: 5년 (전자상거래법)
    • 대금결제 및 재화 등의 공급에 관한 기록: 5년 (전자상거래법)
    • 소비자 불만 또는 분쟁처리에 관한 기록: 3년 (전자상거래법)
    • 웹사이트 방문 기록: 3개월 (통신비밀보호법)
  2. 학습 데이터
    보유 기간: 회원 탈퇴 시까지
    내용: 학습한 표현, 정답률, 숙련도, XP, 레벨, Streak, 배지, 학습 세션 기록
    파기 방법: Supabase PostgreSQL에서 CASCADE DELETE로 즉시 삭제
  3. 익명화된 통계 데이터
    보유 기간: 영구 보관
    내용: 개인 식별 정보가 완전히 제거된 집계 통계 (예: "회의 카테고리 평균 정답률 72%")
    사용 목적: 서비스 개선, 콘텐츠 난이도 조정, 학습 효과 연구
  4. 법령에 따른 보존
    회사는 관계법령에 따라 다음과 같이 개인정보를 보존합니다. 이 경우 회사는 해당 개인정보를 별도의 데이터베이스(DB)로 옮기거나 보관장소를 달리하여 보존합니다.

  • 이용자가 사전에 동의한 경우
  • 법령의 규정에 의거하거나, 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우
※ 현재 회사는 개인정보를 제3자에게 제공하고 있지 않습니다.

제6조 (개인정보의 파기 절차 및 방법)

  1. 파기 절차
    이용자가 회원가입 등을 위해 입력한 정보는 목적이 달성된 후 별도의 DB로 옮겨져(종이의 경우 별도의 서류함) 내부 방침 및 기타 관련 법령에 의한 정보보호 사유에 따라(보유 및 이용기간 참조) 일정 기간 저장된 후 파기됩니다.
    별도 DB로 옮겨진 개인정보는 법률에 의한 경우가 아니고서는 보유 목적 이외의 다른 목적으로 이용되지 않습니다.
  2. 파기 방법
    전자적 파일: Supabase PostgreSQL의 DELETE 쿼리를 통한 영구 삭제, 복구 및 재생이 불가능한 방법으로 파기
    종이 문서: 분쇄기로 분쇄하거나 소각
    Supabase Storage 파일: 프로필 사진 등 업로드된 파일은 Storage API를 통해 완전 삭제
    백업 데이터: 정기 백업된 데이터 내 개인정보도 동일한 절차로 파기
  3. 자동 파기 시스템
    회사는 Supabase RLS(Row Level Security) 정책과 CASCADE DELETE를 통해 회원 탈퇴 시 다음 데이터가 자동으로 파기되도록 설정하고 있습니다:
    • users 테이블: 회원 기본 정보
    • user_progress 테이블: 학습 진행도 데이터 (CASCADE)
    • learning_sessions 테이블: 학습 세션 기록 (CASCADE)
    • Supabase Auth: 인증 정보 및 소셜 로그인 연동 데이터

    개인정보 열람 요구
    오류 등이 있을 경우 정정 요구
    삭제 요구
    처리정지 요구

    앱 내 설정: Profile 탭 → 설정 → 개인정보 관리
    이메일 요청: manager@x-ti.org로 요청서 발송
    처리 기한: 요청일로부터 10일 이내 처리 및 회신
  3. 열람 가능 정보
    앱 내에서 실시간으로 열람 가능한 정보:
    가입 정보: 이메일, 사용자명

    게이미피케이션: 현재 레벨, 총 XP, Streak 일수, 획득 배지
    학습 기록: 최근 7일간 학습한 표현 목록 및 정답률
  4. 만 14세 미만 아동의 권리
    만 14세 미만 아동의 경우 법정대리인이 아동의 개인정보를 조회하거나 수정, 삭제, 처리정지 요구 등의 권리를 행사할 수 있습니다.

제8조 (개인정보의 안전성 확보 조치)

  1. 기술적 조치
    암호화:
    - 비밀번호: bcrypt 해시 알고리즘으로 암호화 저장 (Supabase Auth)
    - 개인정보: PostgreSQL의 TDE(Transparent Data Encryption) 적용
    - 통신 암호화: HTTPS/TLS 1.3 프로토콜 적용
    접근 통제:
    - Supabase RLS(Row Level Security): 사용자는 본인 데이터만 접근 가능
    - IP 화이트리스트: 관리자 DB 접근 시 허가된 IP만 허용
    - 2단계 인증(2FA): 관리자 계정 필수 적용
    침입 탐지:
    - Supabase 보안 모니터링: 비정상 접근 시도 실시간 탐지
    - API Rate Limiting: DDoS 공격 방지
    백업:
    - 데이터베이스 일일 자동 백업 (Supabase Point-in-Time Recovery)
    - 백업 데이터 암호화 저장
  2. 관리적 조치
    개인정보 취급 직원의 최소화 및 교육
    개인정보보호 책임자 지정 및 운영
    내부관리계획 수립 및 시행
    접근권한 관리: 역할 기반 접근 제어(RBAC)
    정기적 보안 감사 실시
  3. 물리적 조치
    Supabase 클라우드 인프라: AWS SOC 2 Type II, ISO 27001 인증 데이터센터 사용
    전산실, 자료보관실 등의 접근통제

제9조 (개인정보 자동 수집 장치의 설치·운영 및 거부)

  1. 자동 수집 장치
    회사는 모바일 앱 특성상 웹 쿠키(Cookie)를 사용하지 않으나, 다음과 같은 자동 수집 장치를 운영합니다:
    - 앱 로컬 스토리지: 로그인 상태 유지, 설정 저장 (AsyncStorage 사용)
    - 푸시 알림 토큰: Expo Notifications를 통해 기기별 고유 토큰 수집
    - 분석 도구: Supabase Analytics (향후 도입 예정, 익명화된 사용 패턴 분석)
  2. 수집 거부 방법
    - 푸시 알림: 앱 설정 → 알림 → 수신 거부
    - 분석 데이터: 앱 설정 → 개인정보 → 데이터 수집 동의 철회
    - 기기 권한: 모바일 기기 설정 → 앱 권한 → 알림 권한 해제
    ※ 다만, 수집을 거부할 경우 일부 서비스 이용이 제한될 수 있습니다 (예: 푸시 알림 거부 시 학습 리마인더 미수신).
